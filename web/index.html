--- about mandelbrot-c ---

a high-performance fractal explorer written in c99.

how it works: the core math is written in c and compiled to webassembly. it uses webgl 2.0 for gpu rendering and emscripten workers for multi-threaded cpu rendering.

web vs desktop app:

  • this web version supports url sharing, screenshots, and 64-bit gpu precision.
  • the desktop app additionally supports: 128-bit deep zoom precision, video recording, 8k mega screenshots, and local bookmarks.
