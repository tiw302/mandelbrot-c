--- about mandelbrot-c ---

a high-performance fractal explorer written in c99.

features:

  • explore mandelbrot, burning ship, tricorn, celtic, and buffalo.
  • interactive julia mode with coordinate tracking (press J).
  • automated tour mode for cinematic exploration (press T).
  • multi-threaded cpu & webgl 2.0 gpu rendering.

web vs desktop app:

  • web: supports url sharing, screenshots, and 64-bit gpu precision.
  • desktop: adds 128-bit deep zoom precision, video recording, 8k mega screenshots, and local bookmarks.

tech stack: c99, webgl 2.0, sokol_gfx, emscripten (wasm)
